# Ronak Thacker — Futuristic Portfolio

Ultra- premium cinematic portfolio for **Ronak Thacker** — AI-Powered Website Developer from India.

## Tech Stack

- **Next.js 16** (App Router)
- **React 19**
- **Tailwind CSS v4**
- **Framer Motion** — scroll & UI animations
- **GSAP** — cinematic loader sequence
- **Three.js + React Three Fiber + Drei** — particles & holographic sphere
- **Lenis** — smooth scrolling

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Structure

1. AI System Boot Loader
2. Cinematic Hero (3D sphere + particles)
3. Developer Dashboard (live status)
4. About
5. Projects Lab
6. Interactive Skills
7. Journey Timeline
8. Why Hire Me
9. Testimonials (infinite marquee)
10. Contact
11. Footer

## Easter Eggs

- **⌘K / Ctrl+K** — Command palette navigation
- **`** (backtick) — Terminal mode (type `hire` → contact)
- **AI** button (bottom-right) — Assistant widget

## Customize

Edit `src/lib/data.ts` — copy, projects, skills, testimonials, email, and social links.

Add `public/resume.pdf` for the Download Resume button.
