# SWAYAM — Cinematic Portfolio

Ultra-premium futuristic portfolio for **SWAYAM** — Video Editor, UI/UX Designer, and Visual Storyteller.

## Tech Stack

- **Next.js 16** (App Router)
- **React 19**
- **Tailwind CSS v4**
- **Framer Motion** — scroll & UI animations
- **GSAP** — cinematic loader sequence
- **Three.js + React Three Fiber** — WebGL particle field
- **Lenis** — smooth scrolling

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Structure

1. Cinematic Loader
2. Hero
3. About
4. Featured Projects
5. Video Showcase (horizontal scroll)
6. UI/UX Showcase
7. Skills & Tools
8. Experience Timeline
9. Testimonials
10. Contact
11. Footer

## Customize

Edit content in `src/lib/data.ts` — projects, skills, timeline, testimonials, and copy.
